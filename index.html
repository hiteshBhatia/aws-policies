Welcome to AWS Policies.

Amazon Web Services provides a beautiful way to control access to specific resources. And for this policies are required, which in some cases could prove to be bit complex to write. I intent to push all the policies (IAM/S3 and others)that I create to this bucket so this becomes a referable directory for all kind of policies.

Caution

Please make sure that you do not commit any real credentials or information related to your Project, as this could have serious side effects.
